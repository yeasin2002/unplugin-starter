# Product Overview

This is a production-ready starter template for creating modern NPM packages with TypeScript. It provides a complete foundation for building, testing, and publishing TypeScript libraries with dual CommonJS and ESM support.

## Purpose

- Serve as a boilerplate for creating new NPM packages
- Provide best-practice tooling and configuration out of the box
- Support both modern ESM and legacy CJS module formats
- Enable automated testing, formatting, and release workflows

## Key Characteristics

- TypeScript-first library development
- Dual module format output (CJS + ESM)
- Comprehensive CI/CD pipeline
- Automated version management with Changesets
